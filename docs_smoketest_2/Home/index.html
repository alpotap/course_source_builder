Documentation

Operations Bridge Manager automatically discovers and monitors the IT infrastructure of an enterprise by collecting topology and event data. You can use event data to resolve infrastructure or service issues. Also, use the topology data to check the health and performance of the IT infrastructure. Operations Bridge Manager framework provides you with a single pane of glass to detect, correlate, solve, and prevent problems occurring in networks, systems, and applications.

You can use OBM for the following:

  • Monitor health and performance of system infrastructure - You can monitor the health and performance of your IT infrastructure by using OBM dashboards. You can customize the workspace and dashboards to filter events and focus on critical issues. You can use agent based Management Packs and third-party connectors to manage IT application and infrastructure domains.
  • Manage events and correlation - You can correlate events from different sources into a central console. This helps you to manage your issues efficiently.
